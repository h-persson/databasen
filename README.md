# Databasen

This project is a relational database built with SQL designed to manage school administration data, including staff, courses, and academic periods.

## To replicate:

1. use create.sql to initialize
2. use insert.sql to seed with data, then insertextra.sql for more data.

### Workload Queries and Optimization 
query1.sql - query5.sql are the workload queries. optimize.sql is for creating materialized view and indices.
